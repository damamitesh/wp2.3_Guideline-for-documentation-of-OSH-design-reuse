       - Pressure valve
       - Flow sensor connector
       - Etc. 
   - Electerical architecture structure  
     - Electrical components name 
       - Carrier Board
       - Sensors 
       - Transistors
       - Etc.
    - Software architectural structure  
      - Software components name

        - Position controller
        - Position and attitude estimator
        - Etc.
