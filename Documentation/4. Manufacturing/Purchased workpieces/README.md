## **What should include the documentation of the purchased workpiece?**

*The bill of marterial (BOM) is the document that describes all the components and there reference. If the component is to be purchased one should find all the information regired to buy the part.*

### *1. Bill of material (BOM)*
<details>
  <summary>Click to see the guideline</summary>
 
 - **Definition:** *A bill of materials (BOM) is a comprehensive list of parts, items, and other materials required to create a product, as well as instructions required for purchasing the required materials and parts.*

```
What should includes the bill of material for the purchased parts(not limited to...)?

   1. Part number
   2. Item name
   3. Description
   4. Seller name
   5. Quantity
   6. Unit price
   7. Extended price according to quantity
   8. Manufacturing standard lead time
   9. Packaging
   10. BOM notes
   11. ...
   ```
</details>
